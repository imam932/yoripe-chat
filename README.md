## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ git clone https://github.com/imam932/yoripe-chat.git
$ cd yoripe-chat
$ npm install
```

## Prepare setting
input setting DB url in file .env

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
